#### Proof Assistants ⅋ Agda
Agda is:
- A powerful programming language (similar to OCaml)
- You can also write proofs (literally any proof)
- Proofs look just like code
- You can write
    + Just code
    + Just proofs
    + Proof about code
    + Code and proofs intertwined (“correct-by-construction”)

#### Class Outcomes

- Mathematic proof and logical reasoning
    + Small and simple core foundation
    + Proving = programming
    + You will gain a deep understanding of principles like induction and
      quantification. This may sound scary, but we will boil these down to a
      few very simple ideas, and we will exercise them through programming in
      the proof assistant.
- The art of specification
    + Necessary and challenging, even for testing
    + It's an art
- The art of verification
    + The spec matters
    + The spec is trusted
    + Some specs which are equivalent in principle may be very different in
      how easy they are to verify. Some specs are stronger than others. Some
      specs are less complex than others. You want the simplest possible spec
      because the spec is trusted—how do you know you got the spec right?
- A final project you can show off (to friends, parents, future employers,
  etc.)
